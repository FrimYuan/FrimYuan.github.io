HEXO操作命令及日常遇到的问题

发表于 2016-11-16 更新于 2020-02-26 分类于 博客
本文字数： 684 阅读时长 ≈ 1 分钟
命令

常用命令

命令 缩写 功能描述
hexo new “postName” hexo n #新建文章
hexo new page “pageName” #新建页面
hexo generate hexo g #生成静态页面至public目录
hexo server hexo s #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
hexo s -p 4001 #开启预览访问端口，使用4001端口号
hexo deploy hexo d #部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

组合命令

命令 功能描述
hexo s -g #生成并本地预览
hexo d -g #生成并上传
hexo s -p 4001 #开启预览访问端口，使用4001端口号

日常问题集合

Mist主题使图片默认居

修改https://github.com/iissnan/hexo-theme-next/blob/master/source/css/_schemes/Mist/_posts-expanded.styl 30行可解决

markdown中需要转义的字符

因为被转义了，所以我在&后面添加了_，实际使用中去掉即可

显示结果 描述 实体名称 实体编号
空格 &_nbsp; &_#160;
< 小于号 &_lt; &_#60;
> 大于号 &_gt; &_#62;
& 与号 &_amp; &_#38;
“ 引号 &_quot; &_#34;
‘ 撇号 &_apos; &_#39;
